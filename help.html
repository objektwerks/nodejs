MadMax is a development tool that allows for creation of named queries that compose a target url and a set of css selectors. It contains the following features:

  • Remote call and connect/disconnect indicators.
  • Add, save and execute query buttons.
  • Seach dialog, listing query names.
  • Search query names via autocomplete, and refresh list (ctrl-r).
  • Query name, non-editable, and query url, editable, executable (return) and savable (ctrl-s).
  • Html and source views, with visual css selector.
  • Tree view of html, with clickable css selector.
  • Css and Js list and source views.
  • Embedded internet browser.
  • Selector source, savable and executable (ctrl-s), execution view and navigation.
  • Executor view of executed query response.
  • View and source views of Executor response.
